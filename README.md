# 📊 Rodricks Otieno Ochieng | Data Scientist & AI Specialist Portfolio

Welcome to the professional portfolio website of **Rodricks Otieno Ochieng**, a results-driven Data Scientist specializing in architecting advanced Machine Learning models, deploying scalable Big Data pipelines, and integrating state-of-the-art Generative AI systems.

This website is custom-tailored to reflect a unique blend of **professional case studies** and **data analytics workflows**, designed entirely in the signature **GitHub Dark theme**.

---

## 🛠️ Technology Stack & Innovations

This application is built with the absolute latest bleeding-edge web technologies:

*   **Framework:** **Next.js 16 (App Router)** - Leveraging Server Components for optimal SEO and rapid loading speeds.
*   **Library:** **React 19** - Utilizing Concurrent Features, streamlined rendering, and modern state transitions.
*   **Styling:** **Tailwind CSS v4** - Implementing utility-first designs entirely configured with new CSS-first `@theme` directives.
*   **Language:** **TypeScript** - Enforcing robust type safety across all metrics, forms, and components.
*   **Icons:** **Lucide React** & custom optimized inline SVGs for professional brand consistency.

---

## ✨ Features & Architecture

The application is structured into modular sections designed to provide a premium, dynamic user experience:

### 1. 🧠 Interactive Neural Network Hero Grid
*   Features a custom-engineered **glowing visual matrix mimicking a GitHub Contribution Grid** that dynamically recalculates active neural node weights.
*   Presents quick statistics highlighting key milestones (5+ Years Experience, 50M+ Data Points, 98% Model Accuracy).
*   Incorporates a smooth command-line-styled header (`neural-network-activity.sh`) and key metrics (GPU Active: 94.2%).

### 2. 📊 Skills & Capabilities Matrix
*   Highlights competencies across **Data Analytics**, **Machine Learning**, **Data Engineering**, and **Full-stack Web Integration**.
*   Displays a curated **Professional Toolbelt** with custom color categorization badges representing Languages, ML models, Cloud Infrastructure, BI Visualizations, and Operations.

### 3. 📂 GitHub Repository-Styled Projects Showcase
*   Presents industry-grade projects in the style of GitHub repositories, complete with Public tags, language dot identifiers, stars, forks, and repository action links.
*   Features structured **Case Study Deep Dives** illustrating clear business impact (e.g. *"14% Customer Churn Reduction Serving FastAPI endpoints"* or *"5,000+ Transactions/Second PySpark pipeline"*).
*   Filters repository cards dynamically by analytical category.

### 4. 📈 Commit-Inspired Experience Timeline
*   Highlights professional history through a vertical chronology designed like a git commit stream.
*   Details responsibilities, technologies deployed, and team management accomplishments.

### 5. ✉️ Settings-Style Communication Hub
*   A sleek, Settings-styled panel built for secure communication, complete with standard validated forms.
*   Includes high-fidelity interactive elements like async submission loaders, animated spinners, and modern success notification banners.

## 🚀 Getting Started

To launch and run the portfolio website locally, follow these steps:

### 1. Clone the repository and navigate in:
```bash
git clone https://github.com/rodricks-ochieng/portfolio.git
cd portfolio
```

### 2. Install dependencies:
```bash
npm install
```

### 3. Spin up the local development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

### 4. Build optimized production bundles:
```bash
npm run build
```

### 5. Run the production build locally:
```bash
npm run start
```
